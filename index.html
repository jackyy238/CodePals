Courses We Offer

Intro to Python

In this course, you'll learn the basics of programming using the popular language Python, which includes fundamentals of programming concepts like variables, data types, conditionals, loops, and functions. Then, you'll dive into using Python to solve some real-world problems. By the end of the course, you'll have a solid foundation in Python and the confidence to tackle more advanced topics on your own. Whether you're interested in a career in software development or just want to learn a new skill, this course is a great starting point for anyone looking to learn programming.

Data Visualisation

In this course, you'll learn how to create effective visualizations that tell a story with your data. You'll dive into the tools and libraries used by data analysts and scientists to create stunning visualizations, such as Matplotlib, Pandas, and Seaborn. These libraries allow you to create a wide range of visualizations, from simple line charts and bar charts to complex heatmaps and interactive dashboards. By the end of the course, you'll have a solid foundation in data visualization and the skills to create compelling visualizations that communicate insights effectively. Whether you're a data analyst, scientist, or business professional, this course is a must-have for anyone who wants to take their data analysis skills to the next level.
